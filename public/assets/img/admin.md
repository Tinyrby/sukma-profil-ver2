Buatkan desain halaman Admin Sistem Informasi Desa (SideS).
Di halaman ini admin desa bisa mengelola data penduduk dan mutasi penduduk.

Fitur yang harus ada:

Tabel Data Penduduk berisi kolom: NIK, Nama, Jenis Kelamin, Tanggal Lahir, Alamat, Status Keluarga, dan Aksi (Edit / Hapus).

Form Tambah Data Penduduk dengan input lengkap (NIK, Nama, Tanggal Lahir, Jenis Kelamin, Alamat, Pekerjaan, Status Keluarga).

Menu Mutasi Penduduk dengan subfitur:

Kelahiran → untuk menambah data bayi baru lahir.

Kematian → untuk menandai penduduk yang meninggal dunia.

Pindah Datang → untuk menambah penduduk baru dari luar desa.

Pindah Keluar → untuk mencatat penduduk yang keluar dari desa.

Tampilkan statistik singkat di bagian atas halaman (misalnya total penduduk, jumlah kelahiran, jumlah kematian, jumlah pindah datang, dan pindah keluar).

Desain tampilannya sederhana, profesional, dan mudah digunakan. Gunakan warna netral biru-hijau khas desa dan tampilan seperti dashboard admin modern.

Pastikan layout responsif agar bisa dibuka di laptop dan ponsel.
